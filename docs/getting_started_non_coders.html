‚öôÔ∏èManual Setup (Advanced Users)

Show manual installation steps

If you prefer to set up manually without conversational guidance:

Step 1: Install Python 3.11+

macOS:

brew install python@3.11

Ubuntu/Debian:

sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update
sudo apt install python3.11 python3.11-venv python3.11-dev

Step 2: Install the Package

Option A: Using Poetry

poetry install

Option B: Using venv

python3 -m venv .venv
source .venv/bin/activate
pip install -e .

Step 3: Initialize Configuration

sce init

Step 4: Configure Strava Credentials

Edit config/secrets.local.yaml with your Strava API credentials from https://www.strava.com/settings/api

Step 5: Authenticate

sce auth url

Follow the authorization URL, approve access, and enter the callback code.

Step 6: Sync Activities

sce sync

Step 7: Create Profile

sce profile create --name "Your Name" --age 30 --max-hr 190

See CLI documentation for complete profile setup options.

Common Issues

"command not found: sce"

Cause: Virtual environment not activated (if using venv) or package not installed

Fix: Ask Claude: "The sce command isn't working" - Claude will diagnose and fix

"python3: command not found"

Cause: Python not installed or not in PATH

Fix: Say: "Help me install Python" - Claude guides platform-specific installation

"Permission denied" during installation

Cause: Virtual environment not activated (trying to install to system Python)

Fix: Claude will detect this and guide you to activate the virtual environment
